# ML-FLow

### Commands used
- ``pip install mlflow``
- ``mlflow server --backend-store-uri mlruns/ --default-artifact-root mlruns/ --host 127.0.0.1 --port 5000``
- ``mlflow run . -P alpha=0.42``
